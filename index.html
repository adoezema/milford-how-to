How-To Guide on Feature Editing

  1. Select "New Feature"
  2. Place pin on desired area of the map.
  3. Fill out any relevant attribute information and select "Create"
  1. Select "Select" undert the Edit Features section
  2. Select desired feature you want to update
  3. Select "Update" when you are finished and want to save your changes
