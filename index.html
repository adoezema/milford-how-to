How-To Guide on Feature Editing

  1. Select the "Editor" button on the top menu bar
  2. Select desired layer you want to add linework to
  3. Enable Snapping
    • You will want to snap lines to points when creating new features
    1. Select "Settings"
    2. Select "Snapping Layers"
    3. Select the "Layer/Group"
    4. Select the desired Layer you want to snap to
  4. Hover over point until red circle appears
  5. Click the start point, should be point feature
  6. Hover over end point until red circle appears
  7. Double Click to finish adding feature
  8. Fill out any relevant attribute information and select "Create"
  1. Select "Select" undert the Edit Features section
  2. Select desired feature you want to update
  3. Select "Update" when you are finished and want to save your changes
